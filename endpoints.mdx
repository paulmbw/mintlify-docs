## Authentication

### Login
```http
POST /auth/login
```

### Register
```http
POST /auth/register
```

## Users

### Get User Profile
```http
GET /users/me
```

### Update Profile
```http
PUT /users/me
```

## Data Operations

### List Items
```http
GET /items
```

### Create Item
```http
POST /items
```
