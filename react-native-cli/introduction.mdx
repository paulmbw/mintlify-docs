With authentication, navigation, and other essential features pre-configured, Launchtoday provides the solid foundation you need to bring your app idea to life.

## Why Choose Launchtoday?

Launchtoday comes pre-configured with everything you need to build a professional mobile app:

### üîê Authentication & Security

- Supabase Auth with social providers (Apple, Google)
- Magic link authentication
- Row Level Security (RLS) policies
- JWT token management and secure storage

### üí≥ Payments & Monetization

- Stripe Payment Sheet with Apple Pay & Google Pay support
- RevenueCat integration for:
  - In-app purchases
  - Subscription management 
  - Consumable items
- Multi-currency support through Stripe

### ‚ö° Backend & Infrastructure

- Supabase PostgreSQL database
- S3-compatible storage through Supabase
- TypeScript Koa API service with:
  - Rate limiting
  - Security middleware


### üé® User Experience

- Push notifications (OneSignal)
- Deep linking
- Dark/light themes
- Multi-language support with RTL
- Responsive layouts
- Pre-built UI components

### üìä Development Tools

- Automated build & release pipeline with:
  - Fastlane for iOS/Android deployment
  - GitHub Actions CI/CD workflows
  - Automated versioning and changelogs
  - App Store & Play Store publishing
- End-to-end testing with Detox
- Analytics and crash reporting
- Beta distribution through TestFlight & Play Console
