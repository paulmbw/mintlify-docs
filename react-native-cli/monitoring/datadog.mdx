## Overview

[Datadog](https://datadoghq.com) provides comprehensive application performance monitoring (APM), logging, and real user monitoring (RUM) for your React Native application.

## Prerequisites

Before beginning the setup, ensure you have:

1. A Datadog account
   - Sign up at [datadoghq.com](https://datadoghq.com)
   - Create a new application
2. Your Datadog API and Application keys
3. Your RUM Application ID
