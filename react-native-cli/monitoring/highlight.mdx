## Overview

[Highlight.io](https://highlight.io) provides session replay, error monitoring, and logging capabilities for your React Native application. This helps you understand user behavior and troubleshoot issues more effectively.

## Prerequisites

Before beginning the setup, ensure you have:

1. A Highlight.io account
   - Sign up at [highlight.io](https://highlight.io)
   - Create a new project
2. Your Highlight.io Project ID
