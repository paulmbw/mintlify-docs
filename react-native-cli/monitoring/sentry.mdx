## Overview

[Sentry](https://sentry.io) provides real-time error tracking and performance monitoring for your React Native application. It helps you identify, triage, and fix crashes and performance issues.

## Prerequisites

Before beginning the setup, ensure you have:

1. A Sentry account
   - Sign up at [sentry.io](https://sentry.io)
   - Create a new project
2. Your Sentry DSN (Data Source Name)
