## Overview

RevenueCat provides:
- In-app purchase management
- Subscription analytics
- Cross-platform receipt validation

## Configuration

### 1. RevenueCat Setup

1. Create account at [revenuecat.com](https://www.revenuecat.com)
2. Create a new app
3. Configure products and subscriptions

### 2. SDK Integration

```typescript
import Purchases from 'react-native-purchases';

Purchases.configure({
  apiKey: 'your_api_key',
  appUserID: 'user_id' // Optional
});
```
