# What is Launchtoday Expo?

Launchtoday Expo is the Expo-based version of our powerful React Native boilerplate. It provides the same carefully crafted features and integrations as our CLI version, but built specifically for Expo-managed workflows.

<Note>
  The Expo version is currently under development. We're working hard to bring
  you all the features available in our React Native CLI version, optimized for
  the Expo ecosystem. Looking for the React Native CLI version? [Click
  here](/react-native-cli/introduction) to check out our React Native CLI
  template documentation.
</Note>
